익스프레스 npm을 활용해서 웹서버 만들기
> http 프로토콜을 활용해서 하드코딩으로 서버 구현
> 노드에서 서버를 구현할 때 사용할 수 있는 라이브러리

노드에서 서버를 구현하는 3대장: 익스프레스 / koa / hapi

https://www.npmtrends.com - 사람들이 많이 사용하는 걸 알려주는 사이트

1. 폴더 생성 후 cd로 터미널 이동
2. npm init -y // package.json 만들기
3. package.json 설정 변경
4. npm i express // express 설치
5. npm i -d nodemon // nodemon 설치

익스프레스는 여러개의 노드패키지를 같이 사용할 수 있다!!
익스프레스는 use()함수와 함께 미들웨어의 역할도 할 수 있다.
react의 상태관리를 효율적으로 도와주는 미들웨어 -> redux

https / express => 아주 간단한 서버만들기
> 복잡한 서버 구현 : 서버에 데이터를 요청하는 클라이언트가 누구냐? 서버가 전달해줘야 하는 값이 달라짐!!
> 로그인 여부 / if문 A/B (조건부) / A: 로그인 > 100명
> 쿠키!!!

쿠키: 서버가 클라이언트의 요청에 따라서 최초에 발급해주는 식별 아이디라고 판단할 수 있음
