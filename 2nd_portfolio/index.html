Project

01#

ChatBot 'Maeil'

1. 개요
챗봇 “매일”은 사용자가 정보를 찾을 때 접근하기 쉽고 더 정확하고 신속한 정보를 제공하기 위해 설계되었습니다.
저희 챗봇 서비스는 1단계 챗봇 서비스로써 패턴 매칭과 키워드 및 연관어 추출을 통해 답변을 제공합니다.
- 학습된 내용에 대한 질의응답
- 사용자와 단순한 형태의 소통
- 질문을 통한 답변 제공

2. 소개
사용자는 서일대학교 관련 정보들을 학교 홈페이지나 블로그 등을 찾아볼 필요없이 챗봇 “매일”을 통해 정확한 정보들을 질문에 따른 답변으로 신속하게 전달합니다.
FAQ ( 자주하는 질문 ) 기능을 통해 타 사용자는 어떤 정보를 필요로 하는지를 확인하는 동시에 버튼 클릭 한 번으로 그에 따른 정보를 얻을 수 있습니다.
사용자마다 각기 다른 질문방식에 따라 키워드 및 연관어를 추출해 그에 맞는 정보를 학습된 내용과 매칭해 답변을 제공합니다.

Project

02#

Mobile Web Page 'Today's MealKit

1. 개요
모바일 전용 페이지 'Today's MealKit는 모바일 쇼핑 페이지를 기반으로 제작한 프론트엔드 페이지입니다.
- 피그마를 사용해 디자인 구성과 레이아웃 설정
- 총 10개의 페이지 구성
- 플러그인을 활용한 이미지 슬라이드와 JavaScript를 사용한 기능

2. 소개
사용자는 하단의 메뉴바를 통해 각 페이지 이동이 용이하며 어느 페이지로 이동하던 같은 메뉴바를 고정시켜놨습니다.
메인 페이지에선 이미지 슬라이더로 구성된 상품들의 정보를 슬라이드하며 삼품들을 볼 수 있으며 각각 다른 메뉴들을 통해 사용자가 원하는 카테고리를 선택할 수 있습니다.

ChatBot 'Maeil'

ChatBot 'Maeil'

챗봇 '매일'은 졸업 작품 팀프로젝트로써 5인으로 구성된 팀원이 만든 프로젝트입니다.
저는 프론트엔드와 로그인,회원가입,게시판의 백엔드를 담당해 작업했습니다.
챗봇을 아이템으로 선정한 이유는 저희 학교의 정보를 보기위한 수단이 부실했기 때문입니다.
따라서 챗봇을 만들어 학생들의 궁금증을 질문하나로 풀어주기 위해 제작했습니다.

  • Area : 서일대학교
  • Completed Date : September 2022
  • Working Day : 4 Month
  • Member : 5

+ 개발환경

개발도구로 Eclipse를 사용하여 톰캣 서버를 설치해 개발했습니다.
언어로 JAVA를 사용했으며 사용자들의 DB는 Maria DB를 통해 관리했습니다.

+ 시스템 구성 및 다이어그램

‘Maeil’은 로그인을 하지 않아도 이용할 수 있는 챗봇, FAQ 기능과 로그인을 해야 이용할 수 있는 게시판, 마이페이지 등의 기능으로 이루어져 있습니다.
메인 화면에서 바로 챗봇과 FAQ를 이용할 수 있고 로그인 후 게시판기능, 마이페이지에서 회원정보 수정, 게시글 조회 등의 기능을 이용할 수 있습니다.
