+ Project Description #02

모바일 웹 페이지 제작

  • 완성일 : 2023 - 03 - 21
  • 소요기간 : 03 - 14 ~ 03 - 21

+ 개발환경

전체적인 사이트의 디자인과 각각의 요소 배치를 Figma를 통해 제작했으며
NotePad를 사용해 코딩 시 class명이 겹치지 않게끔 각 사이트의 이름과 코딩 구성을 설정하고
이를 바탕으로 Visual Studio Code로 사이트를 제작했습니다.

+ 페이지 특징

총 9개의 html 페이지를 만들고 각각 링크를 걸어 페이지 간 연결을 가능하게 했으며
CSS와 JS를 이용해 동작 기능이 가능하게 했습니다.
